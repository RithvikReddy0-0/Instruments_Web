# Demo GIF Workflow

Use this flow to create a concise portfolio demo GIF.

## Storyboard

1. Open the hero and click `Open Playground`.
2. Select `Acoustic Guitar`.
3. Play a few notes.
4. Switch to `Chord` mode and trigger a chord.
5. Start recording, play a short phrase, stop, and replay.
6. Switch to the mobile viewport and show the responsive layout.

## Recommended Capture Settings

- Duration: 12 to 18 seconds
- Width: 1280px desktop, 390px mobile
- Frame rate: 24 fps
- Export: GIF for README, MP4/WebM for portfolio pages

## Suggested File Names

- `screenshots/demo-workflow.gif`
- `screenshots/demo-workflow.mp4`
